One thing to fix before you use this: since you're explicitly inserting id values 1–10, Postgres's auto-increment sequence for that column won't know about them and will still think the next free id is 1. The next row Django creates via the ORM will collide and fail. Reset the sequence right after the insert:
sql

SELECT setval(pg_get_serial_sequence('boxes_box', 'id'), (SELECT MAX(id) FROM boxes_box));
